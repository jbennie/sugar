# isOpera

Detect if is opera

### Example
```js
	import isOpera from 'coffeekraken-sugar/js/utils/is/opera'
if (isOpera()) {
  // do something cool
}
```
Return **{ [Boolean](https://developer.mozilla.org/fr/docs/Web/JavaScript/Reference/Objets_globaux/Boolean) }** true if is opera, false if not
