# isSafari

Detect if is safari

### Example
```js
	import isSafari from 'coffeekraken-sugar/js/utils/is/safari'
if (isSafari()) {
  // do something cool
}
```
Return **{ [Boolean](https://developer.mozilla.org/fr/docs/Web/JavaScript/Reference/Objets_globaux/Boolean) }** true if is safari, false if not
