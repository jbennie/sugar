# isChrome

Detect if is chrome

### Example
```js
	import isChrome from 'coffeekraken-sugar/js/utils/is/chrome'
if (isChrome()) {
  // do something cool
}
```
Return **{ [Boolean](https://developer.mozilla.org/fr/docs/Web/JavaScript/Reference/Objets_globaux/Boolean) }** true if is chrome, false if not
