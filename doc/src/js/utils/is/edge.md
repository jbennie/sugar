# isEdge

Detect if is edge

### Example
```js
	import isEdge from 'coffeekraken-sugar/js/utils/is/edge'
if (isEdge()) {
  // do something cool
}
```
Return **{ [Boolean](https://developer.mozilla.org/fr/docs/Web/JavaScript/Reference/Objets_globaux/Boolean) }** true if is edge, false if not
