# isSamsumgBrowser

Detect if is the samsung stock browser that is running the page

### Example
```js
	import isSamsumgBrowser from 'coffeekraken-sugar/js/utils/is/samsungBrowser'
if (isSamsumgBrowser()) {
  // do something
}
```
