# ```.#{s-component()}--justify```

Provide a simple justify navigation component

### Example
```html
	<ul class="#{s-component()} #{s-component()}--justify">
	<li class="#{s-component()}__item">
 	Item 1
	</li>
	<li class="#{s-component()}__item">
 	Item 2
	</li>
	<li class="#{s-component()}__item">
 	Item 3
	</li>
</ul>
```
