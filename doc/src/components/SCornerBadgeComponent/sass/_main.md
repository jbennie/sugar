## Mixins


### s-corner-badge

Apply the corner badge css


Name  |  Type  |  Description  |  Status  |  Default
------------  |  ------------  |  ------------  |  ------------  |  ------------
$modifiers  |  **{ Number }**  |  The modifiers that represent the corner badge you want. Available : {color} tl tr bl br  |  optional  |  default tr
