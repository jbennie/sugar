# s-translate




Name  |  Type  |  Description  |  Status  |  Default
------------  |  ------------  |  ------------  |  ------------  |  ------------
$x  |  **{ Number }**  |  The x translate value  |  optional  |  0
$y  |  **{ Number }**  |  The y translate value  |  optional  |  0
$z  |  **{ Number }**  |  The z translate value  |  optional  |  0

### Example
```scss
	.my-cool-element {
	@include s-translate(50%, 20px);
}
```
