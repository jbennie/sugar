# s-icon




Name  |  Type  |  Description  |  Status  |  Default
------------  |  ------------  |  ------------  |  ------------  |  ------------
$name  |  **{ [String](http://www.sass-lang.com/documentation/file.SASS_REFERENCE.html#sass-script-strings) }**  |  The icon name to apply  |  required  |

### Example
```scss
	.my-cool-icon {
	@include s-icon(fa-user);
}
```
