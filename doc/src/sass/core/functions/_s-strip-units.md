# s-strip-units

Return a number without any units



### Parameters
Name  |  Type  |  Description  |  Status  |  Default
------------  |  ------------  |  ------------  |  ------------  |  ------------
$number  |  **{ Number }**  |  The number to process  |  required  |

Return **{ Number }** The number without units

### Example
```scss
	s-strip-units(12px); // => 12
s-strip-units(30rem); // => 30
```
