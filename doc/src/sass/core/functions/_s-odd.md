# s-odd

Convert the passed value to the closest odd value



Name  |  Type  |  Description  |  Status  |  Default
------------  |  ------------  |  ------------  |  ------------  |  ------------
$value  |  **{ Number }**  |  The value to convert  |  required  |

Return **{ Number }** The closest odd converted value
