# s-replace-tokens

Search and replace tokens like colors #primary in a passed string, list or map
Supported tokens types :
1. Colors : #{colorName}


### Example
```scss
	s-replace-tokens(12px #primary hello #secondary); // > 12px #ddd hello #fff;
```
